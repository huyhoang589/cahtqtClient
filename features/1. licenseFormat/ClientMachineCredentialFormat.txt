The Registration module (run on the CAHTQT PKI Client machine) produces a JSON credential file that is submitted to the CAHTQT PKI Server:
Client Machine Credential File Format
{
  "token_serial":   "1234567890ABCDEF", // Bit4ID client token serial
  "cpu_id":         "BFEBFBFF000906EA",	// CPUID Processor Serial
  "board_serial":   "SN20240815XYZ01",	// Motherboard factory serial
  "registered_at":  "2025-01-01T00:00:00Z"
}
